## Dashboard Inventario TI

Interfaz gráfica desarrollada en _React_ para visualizar el inventario de hardware de una empresa.

Este proyecto simula la respuesta de un sistema conectado _SQL Server_, mostrando los datos en una interfaz limpia y responsiva.

---

## Características

* Tabla dinámica y responsiva de inventario  
* Búsqueda en tiempo real por equipo  
* Filtrado por categoría  
* Filtrado por estado  
* Indicadores de resumen (Dashboard Cards)
* Interfaz minimalista, limpia y fácil de usar

---

## Tecnologías utilizadas

- React
- Vite
- CSS
- JavaScript

## Instalación
Clonar el repositorio:
git clone https://github.com/ashleyschiaffino/DashboardInventarioTI.git

Entrar al proyecto:
cd DashboardInventarioTI

Instalar dependencias:
npm install

Ejecutar el proyecto:
npm run dev
